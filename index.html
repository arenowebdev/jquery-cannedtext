Lightning fast form letter text entry

Ever have to respond to customers via email using a plain old textarea? This plugin was created to allow you to turn your boring textarea into a typists dream by allowing you to set tab stops on custom tags! Check it out in action below.

Download @ Github »

Show Me The Demo

HINT Try typing "::testing " or "::thanks " (without quotes but with the space) to see the script in action. Once the text fills, the first tag will be highlighted for you. You can then tab to each of the next tabs as necessary.

Example Source

$(document).ready({
    $("#demo").formletter({
        phrases: [
            {key: '::testing', txt: 'Thanks for trying out my first jQuery plugin! As you can tell, I\'m quite the {{LEVEL}} when it comes to making plugins. {{ Another Tag }} just to show it off a bit more.'},
            {key: '::thanks', txt: "{{your name}},\n\nThank you for taking the time to checkout my first plugin.\n\n{{GreeTing}},\n\nDavid Mosher"}
        ]
    });
});
