jQuery Canned Text Input Efficient text entry in form letter textareas with text completion and customizable tab stops

Quick Text
  • Testing
  • Thanks

HELP Try typing "::testing " or "::thanks " (without quotes but with the space) to see the script in action. Once the text fills, the first tag will be highlighted for you. You can then tab to each of the next tags as necessary. You will notice the textarea will scroll to the next tag for you. You will also notice that when there are no more tags, you are able to tab out of the field as you normally would be able to.

Example Usage

$(document).ready({
        $("#demo").cannedtext({
        phrases: [
        {key: '::testing', txt: "Thanks for trying out my first jQuery plugin! As you can tell, I'm quite the {{LEVEL}} when it comes to making plugins. {{ Another Tag }} just to show it off a bit more."},
        {key: '::thanks', txt: "{{your name}},\n\nThank you for taking the time to checkout my first plugin.\n\n{{GreeTing}},\n\nDavid Mosher"}
        ]
    });
    $('.quicktext').click(function(e) {
        e.preventDefault();
        $('#demo').val($('#demo').val() + ' ' + $(this).attr('rel'));
        setTimeout(function() {
            $('#demo').focus().trigger('keydown');
        }, 10);
    });
});
